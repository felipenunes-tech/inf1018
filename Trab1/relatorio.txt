O que foi feito
    - Implementamos as 8 funções da biblioteca BigInt, cobrindo todas as operações de atribuição, aritmética e deslocamento de bits que foram pedidas.
    - Para cada função criada, desenvolvemos uma bateria de testes automatizados para garantir que tudo estava funcionando corretamente.

Testes
    Testamos cada função com valores positivos, negativos e zero, comparando o resultado real com o esperado para validar a operação.

Framework usado
    -CUnit para criar e rodar os testes de forma automática.

Como baixar
    sudo apt update
    sudo apt-get install libcunit1 libcunit1-doc libcunit1-dev

Como compilar e rodar o projeto
    gcc -Wall -o testebigint testebigint.c bigint.c -lcunit
    ./testebigint

Resultado dos testes
    - 100% de sucesso.
